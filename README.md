## 📌 Sobre o Projeto

O **SmartControl IoT** é uma plataforma desenvolvida para controlar e monitorar dispositivos inteligentes em tempo real, como lâmpadas, sensores e fechaduras.

O sistema utiliza uma arquitetura baseada em microserviços, garantindo escalabilidade, organização e segurança.

---

## 🧠 Arquitetura do Sistema

O projeto é composto por múltiplos serviços independentes:

- 🔐 Microserviço de autenticação (Java + JWT)
- 📡 Microserviço IoT (Python + WebSocket)
- 🔀 API Gateway (Node.js)
- 🌐 Frontend Web (HTML, CSS, JavaScript)
- 📱 Aplicativo Mobile (React Native)

---

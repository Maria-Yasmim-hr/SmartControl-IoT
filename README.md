# SmartControl IoT

Sistema completo para controle e monitoramento de dispositivos IoT.

## 🚀 Tecnologias
- Java (Spring Boot)
- Python (FastAPI)
- Node.js (API Gateway)
- HTML, CSS, JavaScript
- React Native

## 🔧 Funcionalidades
- Login com JWT
- Controle de dispositivos
- Dashboard em tempo real
- App mobile
- Arquitetura de microserviços

## 🧠 Arquitetura
Sistema baseado em microserviços com API Gateway centralizando requisições.
